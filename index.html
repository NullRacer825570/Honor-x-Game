# 📸 Giao diện Tải Ảnh & Video (Frontend Demo) Dự án này là một bản demo giao diện (frontend) cho phép người dùng mô phỏng việc tải ảnh và video lên. Mục đích chính là trình bày thiết kế và trải nghiệm người dùng, chứ không phải là một hệ thống tải tệp hoạt động thực sự. **LƯU Ý QUAN TRỌNG:** Vì dự án được triển khai trên GitHub Pages (chỉ hỗ trợ trang web tĩnh), chức năng tải lên **không hoạt động**. Để xây dựng một hệ thống tải tệp đầy đủ, bạn cần một phần backend (như Node.js, Python, PHP) để xử lý và lưu trữ dữ liệu. ## ✨ Tính năng * Giao diện đơn giản, trực quan. * Hỗ trợ chọn nhiều tệp (ảnh hoặc video). * Hiển thị thông báo khi người dùng cố gắng tải tệp lên. ## 🚀 Hướng dẫn sử dụng Chỉ cần mở file `index.html` trong trình duyệt của bạn để xem giao diện. Bạn có thể tải các file này lên một repository trên GitHub và sử dụng GitHub Pages để xem demo trực tuyến.
